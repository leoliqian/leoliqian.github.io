泛型概述

为什么使用泛型
在Java中增加泛型之前，泛型程序设计使用继承来实现的。这样做的安全隐患：

  • 需要强制转换。
  • 可向集合中添加任意类型的对象，存在风险。

泛型的使用

  • 定义集合对象的时候，把泛型写在一对尖括号内（下例前后都是）：
    • List list=new ArrayList();
  • Java SE7及以后的版本中，构造方法中可以省略泛型类型。
  • List list=new ArrayList<>();
